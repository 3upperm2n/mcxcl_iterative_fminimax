# mcxcl_iterative_fminimax
iterative scheme and fminimax function for workload balancing of mcxcl

Test on two platforms
* taote
* zodiac

### Iterative Scheme
Run the following command in matlab.
```matlab
iterPred_taote
iterPred_zodiac
```

### Fminimax
Run the following command in matlab.
```matlab
fminimax_taote
fminimax_zodiac
```
